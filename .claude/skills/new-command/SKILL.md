---
name: new-command
description: Описывает порядок действий для создания новой команды для Discord бота.
---

1. Используй AskUserQuestion чтобы узнать у пользователя:
   - Как команда должна называться (slash-имя, например `play`, `skip`)
   - Что команда должна делать

2. Запусти субагент `developer-command` передав ему полное ТЗ:
   - Имя команды
   - Описание функционала
   - Любые детали, уточнённые на шаге 1

3. Запусти profile "Test Coverage" для покрытия новой команды тестами  

4. Сделай commit изменений в текущую ветку(новая команда + тесты один commit)
